# Pinned to the exact versions verified working on the bioregional.ubec.network
# deployment (2026-07-02). These pins are load-bearing, not cosmetic:
#
#   * fastapi==0.104.1 + starlette==0.27.0 are a MATCHED PAIR. The interface's
#     templates call Starlette's OLD signature TemplateResponse(name, context).
#     Starlette >=1.0 changed this to request-first, which raises
#     "TypeError: unhashable type: 'dict'" on EVERY page. Do not bump starlette
#     independently of fastapi.
#   * anyio<4 is required by fastapi 0.104.1.
#   * pydantic v2 is used; email-validator is needed for the EmailStr fields in
#     the application models, and bcrypt for admin password hashing.
#
# Rebuild the venv reproducibly with:
#     python3 -m venv venv
#     ./venv/bin/pip install -r requirements.txt
