📊 AVAILABLE LAYERS (8 Total)

  Raster Layers:
  1. population_raster_raw              - Raw WorldPop data (grayscale)
  2. population_raster_classified       - Classified density (green gradient)
  3. population_heatmap                 - Heat map style (blue to red)

  Vector Layers:
  4. bioregion_population_boundaries    - Outlines only
  5. bioregion_population_classified    - By total population (7 classes)
  6. bioregion_density_classified       - By density per km² (7 classes)
  7. bioregion_labels                   - Names + population labels
  8. bioregion_population_points        - Centroid points (scaled circles)

🌐 WMS ENDPOINTS

  Base URL:
  https://your-domain.com/cgi-bin/mapserv?map=/opt/mapserver/maps/population.map

  GetCapabilities:
  &SERVICE=WMS&VERSION=1.3.0&REQUEST=GetCapabilities

  GetMap (Single Layer):
  &SERVICE=WMS&VERSION=1.3.0&REQUEST=GetMap
  &LAYERS=population_raster_classified
  &CRS=EPSG:4326&BBOX=-180,-90,180,90
  &WIDTH=1200&HEIGHT=800&FORMAT=image/png

  GetMap (Multiple Layers):
  &LAYERS=population_heatmap,bioregion_labels

  GetFeatureInfo:
  &SERVICE=WMS&VERSION=1.3.0&REQUEST=GetFeatureInfo
  &LAYERS=bioregion_population_classified
  &QUERY_LAYERS=bioregion_population_classified
  &CRS=EPSG:4326&BBOX=-180,-90,180,90&WIDTH=1200&HEIGHT=800
  &INFO_FORMAT=text/html&I=600&J=400

🔧 DEPLOYMENT

  Deploy Mapfile:
  sudo bash deploy_population_map.sh

  Update Database Credentials:
  sudo nano /opt/mapserver/maps/population.map
  # Search for: user='ubec_admin' password='your_password'

  Reload Web Server:
  sudo systemctl reload nginx    # or apache2, or caddy

  Check Permissions:
  ls -la /opt/mapserver/maps/population.map
  # Should be: -rw-r--r-- www-data www-data

🐛 DEBUGGING

  Enable MapServer Debug:
  export MS_ERRORFILE="/tmp/mapserver.log"
  export MS_DEBUGLEVEL=5

  Check Logs:
  tail -f /var/log/mapserver/mapserver.log
  tail -f /var/log/nginx/error.log

  Test Database Connection:
  psql -U ubec_admin -d ubec -h localhost \
    -c "SELECT COUNT(*) FROM phenomenal.population_raster_2025;"

  Validate Mapfile Syntax:
  mapserv -nh "QUERY_STRING=map=/opt/mapserver/maps/population.map&SERVICE=WMS&REQUEST=GetCapabilities"

  Check Spatial Indexes:
  psql -U ubec_admin -d ubec -h localhost -c \
    "SELECT indexname FROM pg_indexes WHERE tablename='population_raster_2025';"

⚙️ COMMON ISSUES

  Issue: Blank images returned
  Fix: Check BBOX matches data extent, verify CRS=EPSG:4326

  Issue: Database connection error
  Fix: Update credentials in mapfile, check pg_hba.conf

  Issue: MapServer not found
  Fix: sudo apt-get install cgi-mapserver mapserver-bin

  Issue: Permission denied
  Fix: sudo chown www-data:www-data /opt/mapserver/maps/population.map

  Issue: Slow rendering
  Fix: Add spatial indexes, enable tile caching, use overviews

🎨 STYLING TIPS

  Adjust Transparency:
  Add &TRANSPARENT=TRUE to GetMap request

  Custom Formats:
  &FORMAT=image/png   (with transparency)
  &FORMAT=image/jpeg  (smaller file size)

  Size Optimization:
  &WIDTH=800&HEIGHT=600   (standard)
  &WIDTH=1920&HEIGHT=1080 (HD)
  &WIDTH=3840&HEIGHT=2160 (4K)
