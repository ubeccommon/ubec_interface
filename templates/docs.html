The Four Elements

Each token represents a fundamental aspect of healthy economic relationships: Air (diversity), Water (reciprocity), Earth (mutualism), Fire (regeneration).

A five-dimensional assessment framework measuring autonomy, ubuntu alignment, reciprocity health, mutualism capacity, and regeneration impact.

Frequently Asked Questions

General Questions

What is the UBEC Protocol?

UBEC (Ubuntu Bioregional Economic Commons) is a four-token economic system built on Stellar blockchain, designed to embody Ubuntu philosophy: "I am because we are." It creates infrastructure where cooperation, reciprocity, and regeneration are economically valuable.

What blockchain does UBEC use?

UBEC tokens operate on the Stellar network, which offers fast transactions (3-5 seconds), extremely low costs (~$0.000003 per transaction), and is carbon-neutral. Stellar is specifically designed for asset issuance and transfer.

How do I get UBEC tokens?

Tokens are distributed through community participation, bioregional engagement, and demonstrating alignment with Ubuntu principles. The protocol rewards diversity, reciprocity, mutualism, and regenerative action.

Technical Questions

What programming languages are supported?

The UBEC Protocol backend is built with Python 3.11+ using FastAPI. We provide an official Python SDK. The API is RESTful, so any language with HTTP support can integrate with UBEC.

Is the code open source?

Yes! The UBEC Protocol is open source under a permissive license. We believe tools for economic liberation should be freely available. Check our GitHub for the source code.

What are the API rate limits?

The public API allows 60 requests per minute per IP address. Higher rate limits are available for registered applications. See the API documentation for details on authentication and rate limit increases.

Community Questions

What is a bioregion?

A bioregion is an area defined by natural characteristics (watersheds, climate, ecology) rather than political boundaries. In UBEC, bioregions are economic communities rooted in specific places, where participants engage in localized reciprocal exchange.

How does governance work?

UBEC uses a progressive decentralization model. Currently in foundation phase with core team stewardship, transitioning to community governance through holonic structures, eventually reaching full decentralization with minimal core team involvement.

Can I start a bioregion in my area?

Absolutely! We encourage bioregional emergence wherever communities are ready. Review the bioregion setup guide, connect with existing coordinators, and reach out for support in establishing your local economic hub.
